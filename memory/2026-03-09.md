## Key Technical Decisions

1. **Tailwind v3 over v4** — v3 is more stable for current projects
2. **Local images** — Better control, instant loading, no CDN latency
3. **Component-based architecture** — Easy client customization
4. **Sonnet-4.6 for heavy lifting** — Used agents for visual improvements & gallery/team work
5. **Two distinct themes** — Dark (barbershop) vs Light (salon)

## Lessons Learned

- Agents (Sonnet) are excellent for bulk visual improvements
- Good documentation is critical for client success
- Local assets > CDN for control & performance
- Component reusability saves enormous amounts of time
- Tailwind v3 is production-safe; v4 needs more maturity
- CSS animations should enhance, not distract
