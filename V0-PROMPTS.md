# V0 Prompts — Loomlab Templates

Use these prompts in v0.app to generate professional website templates.

---

## Template 1: Barbershop / Men's Salon

**Prompt:**

```
Create a modern, premium website for a generic barbershop/men's salon with:

SECTIONS:
- Navbar: logo placeholder, nav links (Services, Gallery, Team, Booking), dark theme toggle
- Hero Section: background image of barber tools/interior, headline "Premium Barbershop", CTA buttons (Book Now, View Services), WhatsApp contact button
- Services Section: grid layout showing 4-6 key services (haircut, beard trim, styling, etc.) with icons and descriptions
- Gallery Section: image grid showcasing recent cuts/styles (use placeholder images), carousel or grid view
- Team Section: barber profiles with names, specialties, and small images
- Testimonials: customer reviews in card format (3-4 testimonials)
- Booking/Schedule Section: simple contact form or booking widget with date/time picker
- Location Section: Google Map placeholder + address, hours, phone, email
- Footer: contact info, social media links, newsletter signup
- Fixed WhatsApp floating button

DESIGN:
- Dark premium aesthetic (dark grays/blacks with gold or emerald accents)
- Professional, modern, clean layout
- Mobile responsive
- Smooth scrolling, subtle animations
- High quality feel suitable for luxury barbershop

OUTPUT: Full Next.js/React component with Tailwind CSS, no hardcoded names (keep generic for reuse)
```

## Template 2: Women's Salon / Manicure

**Prompt:**

```
Create a modern, elegant website for a generic women's salon/manicure studio with:

SECTIONS:
- Navbar: logo placeholder, nav links (Services, Portfolio, Staff, Booking), light theme with toggle
- Hero Section: beautiful, feminine background (spa/salon aesthetic), headline "Beauty & Wellness", CTAs (Book Appointment, Explore Services), WhatsApp button
- Services Section: attractive grid of services (manicure, pedicure, hair, nails, treatments, massage, etc.) with descriptions and pricing
- Portfolio/Gallery: showcase of nail designs, hairstyles, treatments (before/after cards or carousel)
- Staff Section: beauty professionals with photos, specialties, and bios
- Testimonials: client reviews and ratings (4-5 testimonials with photos)
- Special Offers: featured promotions, packages, seasonal deals
- Booking Section: appointment booking form with time slots, services, and contact info
- Location: map, address, hours, contact details
- Footer: info, social links, loyalty program details
- Fixed WhatsApp floating button

DESIGN:
- Light, elegant, feminine aesthetic (soft pastels, pinks, golds, whites)
- Modern & professional but warm & inviting
- Mobile responsive
- Smooth transitions and animations
- Premium spa/salon feel

OUTPUT: Full Next.js/React component with Tailwind CSS, no hardcoded business names (generic/reusable template)
```

---

## Tips for Best Results

1. **Copy the entire prompt** into v0.app chat
2. **Export the code** once generated
3. **Download/unzip** the project
4. **Make small tweaks** (colors, specific services) as needed
5. **Take screenshots** for portfolio

---

## After Generation

Save each template as:
- `templates/barbershop-template/` 
- `templates/salon-template/`

Keep generic — customize per client later.

## Template 1: Barbearia / Salão Masculino

**Prompt:**

```
Cria um website moderno e premium para uma barbearia/salão masculino genérico com:

SECÇÕES:
- Navbar: placeholder logo, links de navegação (Serviços, Galeria, Equipa, Marcação), toggle tema escuro
- Hero Section: imagem de fundo com ferramentas de barbeiro/interior, título "Barbearia Premium", botões CTA (Marcar Agora, Ver Serviços), botão WhatsApp
- Secção de Serviços: grid com 4-6 serviços principais (corte, aparagem de barba, styling, etc.) com ícones e descrições
- Galeria: grid de imagens com cortes/estilos recentes, carrossel ou grid
- Equipa: perfis de barbeiros com nomes, especialidades e fotos
- Testemunhos: avaliações de clientes em cards (3-4 testemunhos)
- Secção de Marcação: formulário simples ou widget com seletor de data/hora
- Localização: Google Maps placeholder + morada, horário, telefone, email
- Rodapé: informações de contacto, redes sociais, newsletter
- Botão WhatsApp flutuante fixo

DESIGN:
- Estética escura e premium (cinzentos escuros/pretos com acentos dourados ou esmeralda)
- Layout profissional, moderno e limpo
- Responsivo para móvel
- Scroll suave, animações subtis
- Sensação de luxo adequada para barbearia premium

OUTPUT: Componente Next.js/React completo com Tailwind CSS, sem nomes fixos (genérico para reutilizar)
```

## Template 2: Salão Feminino / Manicure

**Prompt:**

```
Cria um website moderno e elegante para um salão feminino/manicure genérico com:

SECÇÕES:
- Navbar: placeholder logo, links (Serviços, Portefólio, Equipa, Marcação), tema claro com toggle
- Hero Section: fundo bonito e feminino (estética spa/salão), título "Beleza & Bem-estar", CTAs (Marcar Consulta, Explorar Serviços), botão WhatsApp
- Secção de Serviços: grid atrativo de serviços (manicure, pedicure, cabelo, unhas, tratamentos, massagem, etc.) com descrições e preços
- Portefólio/Galeria: showcase de designs de unhas, penteados, tratamentos (cards antes/depois ou carrossel)
- Equipa: profissionais de beleza com fotos, especialidades e bios
- Testemunhos: avaliações de clientes com ratings (4-5 testemunhos com fotos)
- Ofertas Especiais: promoções em destaque, pacotes, deals sazonais
- Secção de Marcação: formulário de marcação com slots de tempo, serviços e contacto
- Localização: mapa, morada, horário, contactos
- Rodapé: informações, redes sociais, programa de fidelização
- Botão WhatsApp flutuante fixo

DESIGN:
- Estética clara, elegante e feminina (tons suaves, rosas, dourados, brancos)
- Moderno & profissional mas quente & acolhedor
- Responsivo para móvel
- Transições suaves e animações
- Sensação premium de spa/salão

OUTPUT: Componente Next.js/React completo com Tailwind CSS, sem nomes de negócio fixos (template genérico/reutilizável)
```
